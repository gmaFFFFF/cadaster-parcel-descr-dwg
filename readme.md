# Описание
## Общие сведения
Программа «Описание» – это небольшое приложение, работающее совместно с Microsoft Excel и AutoCAD. 
Предназначена для:
1. Автоматической нумерации поворотных точек границы земельного участка в автокаде, с учетом существующих точек;
2. Подготовки некоторых разделов Описания земельных участков (форма утверждена приказом Федеральной службы земельного кадастра России от 2 октября 2002 г. № П/327 «Об утверждении требований к оформлению документов о межевании, представляемых для постановки земельных участков на государственный кадастровый учет»): 
- чертеж земельных участков; 
- описание границ, 

## Инструкция
### Режим подготовки «заготовок» для Описания земельных участков
#### Результат операции: 
- файл Excel содержащий сведения о вновь образованных и прекращающих существование точках и участках границ; 
- геодезические данные на участок; 
- ведомость вычисления площади земельного участка; 
- файл AutoCAD, содержащий заготовку раздела Чертеж земельных участков.

#### Порядок действий: 
1. Сформируйте файлы содержащие каталоги координат точек в формате:

номер_точки1, координата1, координата1
…

номер_точкиN, координатаN, координатаN

2. Укажите каталог, куда Вы желаете поместить результаты расчетов
3. Перенесите файлы (лучше сразу несколько, так будет быстрее чем по одному) в главное окно программы.

![Главное окно](/doc/img/mainWindow.png)

### Режим нумерации участка
#### Результат операции: 
Каталог координат межевых знаков земельного участка на основании: полилинии в автокаде, представляющей собой границу участка, а также существующих точек.

#### Порядок действий: 
1. Откройте Автокад и создайте в нем границу участка в виде полилинии
2. Определите в чертеже блок «Точка» (рекомендую скопировать из файла шаблона)
3. Разместите блоки «Точка» на существующих точках и присвойте им соответствующие номера.
4. Нажмите кнопку DWG
5. Нажмите кнопку нумеровать и щелкните по полилинии – участок будет пронумерован

![Окно нумерации](/doc/img/numWindow.png)

6. Если Вы не указали какую-то из существующих точек, то измените необходимый атрибут номер_точки на номер существующей точки и нажмите кнопку Уточнить имена

### Режим черчения кадастровой сетки
#### Результат операции: 
Кадастровая сетка.

#### Порядок действий: 
1. Нажмите в главном окне на кнопку с решеткой. 
2. В окне "Кадастровая сетка" Вы можете выбрать способ определения границы участка:
    - указать вручную полилинию, представляющую границу участка, для которого необходимо вычертить сетку (это может быть даже полилиния, проходящая из нижнего левого угла до правого верхнего угла)
    - позволить программе самой определить границу по блокам "Точка", расположенным в чертеже. 

![Окно черчения сетки](/doc/img/gridWindow.png)

3. Масштаб подписей сетки Вы можете или установить вручную, или можете позволить программе самой определить оптимальный масштаб. 
4. Нажать кнопку Чертить -- в зависимости от сделанных настроек программа предложит указать границу участка и кадастровый квартал, либо только кадастровый квартал.
